# homebrew-macwhisperauto

Homebrew tap for [MacWhisperAuto](https://github.com/chrisns/MacWhisperAuto) — automatic meeting detection and MacWhisper recording for macOS.

## Install

```bash
brew install --cask chrisns/macwhisperauto/macwhisperauto
```

## Update

```bash
brew upgrade --cask macwhisperauto
```

## Uninstall

```bash
brew uninstall --cask macwhisperauto
```

## Note

MacWhisperAuto is ad-hoc signed and not notarized. On first launch, macOS Gatekeeper will block it. Allow it in **System Settings > Privacy & Security**.

Requires [MacWhisper](https://goodsnooze.gumroad.com/l/macwhisper) to be installed separately.
